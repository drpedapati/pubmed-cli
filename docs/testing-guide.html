pubmed-cli Testing Guide

Release-oriented manual test plan for the non-AI command set.

Prerequisite: set an API key for smoother testing: export NCBI_API_KEY="your-key"

Core Happy Paths

pubmed --help
pubmed search "autism" --limit 2 --human
pubmed fetch "38000001,38000002" --json
pubmed cited-by 38000001 --limit 2 --human
pubmed references 38000001 --limit 2 --human
pubmed related 38000001 --limit 2 --human
pubmed mesh "depression" --human

Validation Paths (Must Fail Cleanly)

pubmed related 38000001 --limit -1
pubmed search autism --sort newest
pubmed search autism --year 2025-2020
pubmed fetch abc123

Expected Outcomes

CaseExpected
Invalid limitError message, no panic/crash
Invalid sortError lists valid values: relevance/date/cited
Invalid yearError explains format/range issue
Invalid PMIDError indicates PMIDs must be digits
Comma-separated PMIDsFetch succeeds with parsed IDs

CSV/JSON Export Checks

pubmed search "autism" --limit 3 --csv /tmp/search.csv --json
pubmed fetch 38000001 38000002 --csv /tmp/fetch.csv
